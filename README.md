## `// MOTION ARCHITECTURE`

The entire site runs on a **global motion system** тАФ not random animations sprinkled around.

```
lib/
тФФтФАтФА motion/
    тФЬтФАтФА variants.ts        # Shared animation presets
    тФЬтФАтФА scroll.ts          # GSAP + ScrollTrigger logic
    тФЬтФАтФА transitions.ts     # Page transition configs
    тФФтФАтФА lenis.ts           # Smooth scroll bootstrap
```

**How it all connects:**

```
Lenis (smooth scroll)
  тФФтФАтФА GSAP ScrollTrigger (scroll timelines & reveals)
        тФФтФАтФА Framer Motion (component & page transitions)
              тФФтФАтФА CSS transforms only (60fps guaranteed)
```

- All animations use **`transform` + `opacity`** тАФ zero layout thrashing
- ScrollTriggers are **cleaned up on unmount** тАФ no memory leaks
- **Reduced-motion fallback** baked in for accessibility
