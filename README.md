# Minhaj — The Motion Architect

A futuristic, motion-driven portfolio built with **Next.js App Router**.  
Designed to feel alive through cinematic animations, glassmorphism UI, and engineered micro-interactions.

This is not a static website.  
It’s a **motion experience**.

---

## Concept

This portfolio focuses on **how interfaces should feel**, not just how they look.

- Fluid smooth scrolling
- Cinematic section reveals
- Interactive glass cards
- Magnetic buttons & micro-interactions
- Premium page transitions
- Ambient background motion

Everything is intentional, minimal, and futuristic.

---

## Tech Stack

| Tech | Purpose |
|---|---|
| Next.js (App Router) | Framework & routing |
| Tailwind CSS | Styling & design system |
| Framer Motion | Page & component transitions |
| GSAP + ScrollTrigger | Scroll animations |
| Lenis | Smooth scrolling engine |

---

## Motion System

A global motion architecture powers the entire site:

- Lenis for buttery smooth scroll
- GSAP timelines for scroll reveals
- Framer Motion for transitions
- Transform + opacity only (60fps)
- Reduced-motion support for accessibility

---

## Pages

- **Home** — Cinematic hero & stack showcase
- **Work** — Animated project grid with parallax
- **Services** — Layered glass UI with motion flow
- **Academic** — Scroll-drawn timeline
- **Contact** — Interactive call-to-action
- **Not Found / Error / Loading** — Fully animated system pages
